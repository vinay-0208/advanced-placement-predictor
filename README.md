# 🎓 Advanced Placement Predictor

A Machine Learning web application built with Python and Streamlit to predict whether a student is likely to get placed based on academic and skill-related factors.

---

## 🚀 Features

- Placement prediction using ML
- Interactive Streamlit dashboard
- Real-time prediction probability
- Feature importance analysis
- Clean visual interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly

---

## 📊 Input Features

- CGPA
- Aptitude Score
- Communication Score
- Projects Completed
- Internship Experience

---

## 🎯 Output

- Placed / Not Placed
- Placement Probability
- Model Insights
